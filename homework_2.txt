DNS - это сервер, который хранит базу данных для доступа компьютера к страницам в интернете. 
Данные для доступа к веб-странице  представляются ip-адресом и доменным именем. Так как компьютер не понимает букв,
доменное имя содержит цифры (код для обмена информацией компьютера  с сервером), которые соответсвуют определенному 
ip-адресу (например google.ru, yandex.ru). То есть DNS связывает буквы и цифры, определяет соответствующий цифровой код для доступа к сайту. 
С помощью DNS компьютер понимает на какую веб-страницу ему следует перейти.
